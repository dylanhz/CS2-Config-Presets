## Files



| 使用场景          | 文件名称              |
|:-------------:|:-----------------:|
| 游戏设置          | `auto.cfg`        |
| 准星参数    | `crosshair.cfg`   |
| 投掷物准星          | `crosshair_throw.cfg`    |
| 跑图指令          | `practice.cfg`    |
| 启动项          | `boot_option.txt`    |

## Path



```
...steamapps\common\Counter-Strike Global Offensive\game\csgo\cfg
```

## Use
- Boot option:
```
+exec auto.cfg
```
- In-game console:
```
exec auto // crosshair included in auto
exec practice
```
